# Amalgadon Board Capture — HDT Plugin

A [Hearthstone Deck Tracker](https://hsreplay.net/hdt/) plugin that reads your current Battlegrounds board and opens it in [Amalgadon](https://www.amalgadon.com) — pre-filled and ready to annotate or share.

---

## Requirements

- [Hearthstone Deck Tracker](https://hsreplay.net/hdt/) (v1.51 or later)
- Windows

---

## Installation

- Download **`AmalgadonPlugin.dll`** from the [latest release](../../releases/latest).
- Copy it to your HDT plugins folder: \`%AppData%\\HearthstoneDeckTracker\\Plugins\\\`
- (Re)start Hearthstone Deck Tracker.
- Enable the plugin in Options -> Tracker -> Plugins

Alternatively, you can **drag and drop** the downloaded plugin zip folder, or extracted dll file into the Options -> Tracker -> Plugins window. You'll then have to restart HDT, and enable the plugin in Options -> Tracker -> Plugins.

---

## Usage

While in a **Battlegrounds game**, click the **"Open in Amalgadon"** button that appears in the HDT overlay.

Your board — hero and all seven minion slots — will open in Amalgadon instantly, ready to annotate and share.
